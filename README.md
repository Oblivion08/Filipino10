# Pamana ni Simoun: Sounds + Characters Version

Ito ang updated version na may:
- Character selection
- NPC dialogue system
- Background music
- Sound effects
- Achievement popup
- Mini-games per topic

## Important folders

assets/music/
- elfili.mp3

assets/sounds/
- click.mp3
- correct.mp3
- wrong.mp3
- victory.mp3
- achievement.mp3

assets/characters/
- optional images kung gusto mo pang palitan ang emoji characters later

## How to add your El Fili sound

1. Ilagay ang music file mo sa:
   assets/music/

2. Rename it as:
   elfili.mp3

3. Open index.html sa browser or Live Server.

## Kung wala pang sound effects

Okay lang. Gagana pa rin ang game. Pero walang maririnig hanggang lagyan mo ng mp3 files.
